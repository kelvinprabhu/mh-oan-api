# How to Register and apply process for individual Beneficiary under Nanaji Deshmukh Krishi Sanjivani Prakalp 2.0?
### 3.0 Registration Process
### 3.1 Farmer Login with AgriStack Farmer I.D.
Beneficiary farmers under the project are not required to register and can log in directly using the Farmer I.D. number. If the farmers do not have a Farmer I.D. number, it is mandatory for the beneficiary to go to the AgriStack portal, register, and obtain a Farmer I.D. number.
### 3.2 Registration process for individuals from landless families
The registration process on the DBT portal is mandatory only for individuals from landless families under the project. Therefore, these beneficiaries should use the website https://dbt-ndksp.mahapocra.gov.in/ or the Mahavistar A.I. application for registration. After visiting the project website, the home page will appear as follows:
Description: After accessing the project website, the initial screen for beneficiary registration for individuals from landless families will appear as above. Click on the green icon next to the bold heading "Farmer" on the left side. click on the "Register" tab in the box on the right side for individuals from landless families to register. Then, a pop-up screen showing information before registration will appear first.
The list of required documents and mandatory items for registration should be kept ready. For example, an active mobile number, Aadhaar number (must be linked with Aadhaar), and if necessary, scanned copies of Caste Certificate / Disability Certificate / Divorcee or Widow Certificate for women, etc., should be available. After that, upon clicking the "Proceed to Register" button, the first registration homepage will appear.
The registration process is to be completed in four stages: Mobile Number Verification, Aadhaar Number Verification, Other Information, and Self-Declaration.
Step 1: First enter your active mobile number. Then, an arithmetic example is given below; enter its correct answer (captcha) in the provided box. Next, click on the "Get OTP" button. A six-digit OTP will be received on the entered mobile number. If there is a delay in receiving the OTP within the specified time, you can click again to get the OTP. Then, enter the received OTP correctly in the given box and click on the "Verify" button. A message indicating successful verification of the beneficiary's mobile number will appear on the screen.

Step-2: First enter your Aadhaar number. Then, select the "OTP" option for authentication. An OTP will be received on the beneficiary's Aadhaar-linked mobile number. If there is a delay in receiving the OTP within the specified time, you can click to get the OTP again. Then, enter the received OTP correctly in the given box and click on the "Verify OTP" button. A message indicating that the beneficiary's Aadhaar number has been successfully verified will appear on the screen. The beneficiary's Aadhaar details will also be displayed along with a photo. After that, click on the "Proceed" button.
Step-3: The address as per Aadhaar will be displayed first. You need to enter your residential address in it. Select the state, district, taluka, and village from the given list and enter the pin code in the box below. Next, the caste category is given; select the option from it. If the beneficiary is from Scheduled Castes/Scheduled Tribes, a screen to upload the caste certificate will become available. On that, select the PDF file of the caste certificate with a maximum size of 5 MB and click on the upload button.
A special category for women is provided. If applicable, select the option for divorcee or widow. Then, a screen to upload the relevant certificate will become available. On that, select the PDF file of the certificate with a maximum size of 5 MB and click on the upload button.
Next, disability information needs to be provided. If the beneficiary has a disability, a screen to upload the disability certificate will become available. On that, select the PDF file of the disability certificate with a maximum size of 5 MB and click on the upload button.
Then, information about the individual from a landless family needs to be entered. In this, first select the district, taluka, and village where the beneficiary is a resident of the project village. Then, the certificate of being landless needs to be uploaded. In the screen below that, select the PDF file of the landless certificate with a maximum size of 5 MB and click on the upload button. Finally, click on the "Proceed" button.
# Step-4:
The text as per the project's guidelines in the self-declaration form will be displayed first. Read and understand it. If you agree with the project's terms and conditions, finally select the given empty box. Then, click on the "Review" button. After that, the information entered during registration will be displayed in a summarized format. After confirming it, click on the "Login" button. The beneficiary's project dashboard will then open. Also, if the beneficiary does not agree to the terms and conditions, they can click on the "I do not agree" button without clicking the checkbox.

# 4.0 Application Process
Registered farmers/beneficiaries under the project should use the website https://dbt-ndksp.mahapocra.gov.in/ or the Mahavistar A.I. application to process applications for personal benefit components on the project's DBT portal.
### 4.1 Farmer Login with AgriStack Farmer I.D.
Beneficiary farmers under the project should log in directly using the AgriStack Farmer I.D. number. If the farmers do not have a Farmer I.D. number, it is mandatory for the beneficiary to go to the AgriStack portal, register, and obtain a Farmer I.D. number.
After visiting the project website, the home page will appear. The farmer beneficiary should first select the "AgriStack Farmer ID" option from the authentication types under the Farmer tab on the "Login" page. Then, enter the 11-digit Farmer ID in the given blank box. After that, an arithmetic example is given below; enter its correct answer (captcha) in the box provided and click on the "Verify" button. A six-digit OTP will be received on the Aadhaar-linked mobile number. If there is a delay in receiving the OTP within the specified time, you can click to get the OTP again. Then, enter the received OTP correctly in the given box and click on the "Login" button. After the Aadhaar and OTP verification is complete, the farmer's dashboard screen will be appeared.
### 4.1.1 Farmer Profile
After logging into the personal benefit DBT portal, it is mandatory for the farmer beneficiary to first update the "Profile". For this, select the "Profile" tab from the main menu on the left side of the dashboard screen. In this, it is necessary to update personal information, address, and farm land information.
Step-1: When updating the profile, first verify the personal information. Then, an OTP-based procedure for updating the mobile number is provided. Next, disability information needs to be provided. If the farmer beneficiary has a disability, a screen to upload the disability certificate will become available. On that, select the PDF file of the disability certificate with a maximum size of 5 MB and click on the upload button.
Next, the caste category is given; select the option from it. If the beneficiary is from Scheduled Castes/Scheduled Tribes, a screen to upload the caste certificate will become available. On that, select the PDF file of the caste certificate with a maximum size of 5 MB and click on the upload button.
A special category for women is provided. If applicable, select the option for divorcee or widow. Then, a screen to upload the relevant certificate will become available. On

that, select the PDF file of the certificate with a maximum size of 5 MB and click on the upload button. Finally, click on the "Proceed" button.
Step-2: The beneficiary should confirm that the displayed address linked to the Aadhaar number is correct.
Step-3: The farm land information section shows the details of the farmer beneficiary's land. It displays the details of the land within the project village as well as the details of the land outside the project village. If there are joint account holders, their details are also shown for informational purposes. Before applying, the farmer beneficiary must click on the "Edit" link and enter the water source and energy source. After that, the farmer beneficiary can apply for various components for personal benefit.
# 4.1.2 Application process for a new component
After the farmer beneficiary updates the "Profile", select the "Apply for New Component" tab from the main menu on the left side of the dashboard screen. Upon entering, the application process is to be completed as follows.
## Step-1: Select Component Name
When selecting a component for personal benefit, first select the "Component Group" from the displayed Drop-Down list. Then, select the proposed "Component Name". Next, if there is a "Type/Crop/Season" related to the component, select the required option from the given Drop-Down list. Also, if a unit is applicable to the relevant component, select the "Unit" (e.g., spacing, dimensions, size, etc.). The consolidated details of the selected item by the farmer beneficiary are mentioned on the right side under "Component Details". After confirming it, click on the "Proceed" button at the bottom right.
## Step-2: Eligibility Criteria and Required Documents for the selected component
After the farmer selects the component, on the next page, as per the project guidelines for the component, information about the beneficiary's eligibility criteria and the necessary documents to be uploaded with the application on the DBT portal will be available on the screen. If there are any changes in the selected component use the "Go Back" button. After confirming the information provided on this page, click on the "Proceed" button at the bottom right of the screen.
The details of the farmer beneficiary's land in the project village and land outside the project village, as well as the total land (Ha. Are), are shown for information. The "Village" where the component is to be implemented and benefit is to be taken should be selected first. Then, the "Survey Number" from the said village where the benefit is to be taken should be selected from the given Drop-down list.

The details of the selected survey number will be available on the screen. It will show the survey number's farm ID, account type, and total area (Ha. Are). Also, if benefit has been taken for other area-based components or the same component from other government schemes or the current project, that area will be shown in "Benefitted Area (Ha. Are)". Also, the area for which an application is currently being made for the same survey number will be shown in "Area in Process (Ha. Are)". Then, the "Eligible Area for Component (Ha. Are)" will be displayed by the system. Next, enter the proposed area for the component (Ha. Are) in the given box.
If the applicant is a joint account holder, upload the PDF file of the consent letter from other account holders. A sample for that is provided, which should be downloaded. If other documents are required for a component as per the guideline criteria, upload the PDF file of those documents. After that, click on the "Proceed" button at the bottom right of the screen.
As shown above, a pop-up screen will be available to verify the information entered by the farmer beneficiary. If any corrections or changes are to be made in the said information, use the "Edit" button. After verifying the information, click on the "Proceed to Apply" button at the bottom right of the pop-up screen.
# Step-4: Undertaking
Undertaking for the proposed component will appear on the next screen. If the terms and conditions in it are agreeable, select the checkbox given at the end of the screen and click on the "Proceed" button.
## Step-5: Self-Declaration
A self-declaration will appear at the end. If the farmer agrees with the mentioned terms and conditions of the project, select the checkbox in front of each term and condition and click on the "Submit" button.
After that, a message indicating the successful submission of the application will appear on the next screen. A button is provided to download or print the said application. Click on it and save the application.
### 4.1.3 My Applications
The farmer beneficiary should click on the "My Applications" tab in the main menu on the dashboard. This will show the application's current status and the level at which the application is. Also, to see the application details, click on the "View" link.
### 4.1.4 Payment Request
The farmer beneficiary will receive pre-sanction from the field level to implement the component under the project. After that, once the farmer has actually implemented the component, as shown above, click on the "Payment Request" tab in the main

menu. To see the application details, click on the "View" button and you can see the application.
After the implementation of the component, click on the "Payment Request" button next to the component for which the grant is to be requested.
If there are required documents related to the component (e.g., plan, certificates, and work completion proof), select and upload their PDF files. After that, if bills are required, record the bill details and upload its PDF file. In this, the bill number, bill date, and amount should be entered accurately.
If there is more than one bill, click on the "Add Bill" button provided at the bottom left and enter the bill details separately. After that, click on the "Payment Request" tab at the bottom right of the screen.
After the payment request, you can check the status of your application from time to time by clicking on the "Payment Request" tab to see the processing level.
# 4.2 Login for individuals from landless families
Registered individuals from landless families under the project have to log in using their own Aadhaar number. It is mandatory for the beneficiary to first register on the DBT portal before logging in. After visiting the project website, the home page will appear.
The beneficiary should first select the "Aadhaar" option from the authentication types under the Farmer tab on the "Login" page. Then, enter the 12-digit Aadhaar number in the given blank box. After that, an arithmetic example is given below; enter its correct answer (captcha) in the box provided and click on the "Verify" button. A six-digit OTP will be received on the Aadhaar-linked mobile number. If there is a delay in receiving the OTP within the specified time, you can click to get the OTP again. Then, enter the received OTP correctly in the given box and click on the "Login" button. After the Aadhaar and OTP verification is complete, the beneficiary's dashboard screen will be available.
### 4.2.1 Profile for individuals from landless families
After logging into the DBT portal, the beneficiary should first update the "Profile" if necessary. For this, select the "Profile" tab from the main menu on the left side of the dashboard screen. In this, you can update personal information, address, and project location information.
When updating the profile, first verify the personal information. Then, an OTP-based procedure for updating the mobile number is provided. If the beneficiary has a disability, they can update the disability certificate. On that, select the PDF file of the disability certificate with a maximum size of 5 MB and click on the upload button.

Next, a caste category is given, in which if the beneficiary is from Scheduled Castes/Scheduled Tribes, they can update the caste certificate. On that, select the PDF file of the caste certificate with a maximum size of 5 MB and upload it.
A special category for women is provided. If applicable, the option for divorcee or widow can be updated, and the relevant certificate can be uploaded. The maximum size for the PDF file of the certificate is 5 MB , which should be selected and uploaded by clicking the button. Finally, click on the "Proceed" button.
# Step-2: Address Information
The beneficiary should confirm that the displayed address linked to the Aadhaar number is correct.
## Step-3: Project Location Information
In the project location information, the details of the beneficiary's village in the project area are displayed.
